### Setup
1. git clone --recursive [path]
2. create new repo and switch origin away from this template -> git remote set-url origin <new_repo_url>
3. cd godot-cpp and checkout current Godot version
4. cd godot-cpp and scons
5. rename project scons and "game" folder if desired
6. run project scons
7. create new Godot project inside the "game" folder (ignore warning for not empty)
